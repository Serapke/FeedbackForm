# FeedbackForm

Simple feedback form application based on given requirements.
Created using Ruby on Rails, PostgreSQL database.

Requirements:
- Comment can be made by both registered and unregistered user.
- If comment is made by unregistered user, user has to fill in his details (phone, email, name).
- If comment is made by registered user, the details are filled automatically from the database.
- Admin can delete and edit comments.
- Comments are shown by 10 in page and sorted by date.
